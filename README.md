# Sudoku

Small Rust command to solve Sudokus.

Compile it with [cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html):

```
cargo build --release
```

## Use

```
cargo run -- -f sample/easy.txt
```

or

```
sudoku -f sample/difficult.txt
```
